Voice Lessons

Browse a catalog of singing lessons and, optionally, join a class.

This catalog is exposed via a REST endpoint that returns the catalog of courses as an XML document.

Two implementations

This app demonstrates two ways to handle XML data in an HTML document:

  • Apply an XSL stylesheet to the XML, using the web browser's built-in rendering engine, or
  • Convert the XML to JSON on the server, when a standard HTML document is requested1.

XML Transformation

Raw XML data is transformed into an XHTML-like document according to the XSL stylesheet applied to it. The result is viewed here.

Server-generated web page

Special HTMX attributes are inseerted into a standard1 HTML5 document so that a server application endpoint2 is invoked. Here is the (alternate) web page.

1A special AJAX library called HTMX is used to invoke endpoints exposed by the servrer application.

2Server application was written with MojoJS for the NodeJS programming language.
